# Personal Portfolio Website

A modern, responsive portfolio website built to showcase my skills, projects, and professional journey as a Frontend Developer. This portfolio serves as a central platform where recruiters, clients, and collaborators can explore my technical expertise, featured projects, and contact information through a clean and engaging user experience.

The website is designed with a strong focus on responsiveness, performance, accessibility, and modern UI principles. It demonstrates my ability to create visually appealing and user-friendly web applications using current frontend technologies.

## 🚀 Features

* Fully responsive design for desktop, tablet, and mobile devices
* Modern and visually appealing user interface
* Interactive hero section with professional introduction
* About section highlighting skills, experience, and goals
* Projects showcase with project details and live links
* Skills section displaying technical competencies
* Smooth animations and user interactions
* Contact section for easy communication
* Optimized for performance and accessibility
* Clean and intuitive navigation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Tailwind CSS
